# Boutique Hostel - Landing Page

## Descripción del Proyecto
Proyecto personal desarrollado como propuesta comercial proactiva para un hostal boutique. El objetivo es crear una *landing page* puramente informativa, enfocada en la conversión directa vía WhatsApp o llamada telefónica. 

El diseño busca transmitir una estética "Boutique Vibes" (tranquilidad, elegancia, minimalismo), utilizando una paleta de colores cálidos (terracota, taupe, blancos rotos) y una tipografía editorial.

## Stack Tecnológico
* **Frontend:** React + Vite
* **Lenguaje:** JavaScript
* **Estilos:** Tailwind CSS (v3) + `@tailwindcss/forms` + `@tailwindcss/container-queries`
* **Animaciones (Fase 2):** GSAP (ScrollTrigger)
* **Despliegue:** Sitio estático (Netlify/Vercel) - *Sin backend*

## Fases de Desarrollo
1.  **Fase 1 (Actual):** Migración del prototipo estático (HTML/Tailwind) a una arquitectura de componentes React. Ajuste fino del diseño responsivo (Mobile-First) respetando el sistema de diseño estricto.
2.  **Fase 2:** Inyección de interactividad avanzada con GSAP, destacando un efecto inmersivo de "Hero Scroll" (zoom-in suave al hacer scroll).
